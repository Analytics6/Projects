# Serverless Data Processing with Kubernetes Orchestration

Brief: Combine serverless compute with Kubernetes for scalable data workloads.

- Objectives:
  - Optimize cost and scalability for batch/stream workloads.
- Scope:
  - Serverless functions, event-driven processing, K8s orchestration.
- Key Components:
  - FaaS, K8s CronJobs, event buses.
- Technologies:
  - Azure Functions, AKS, Knative, Kafka.
- Deliverables:
  - Patterns, templates, cost estimates.
