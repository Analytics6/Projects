# Dacker Brands – Global Integration & Reporting

Brief: Global integration and reporting platform for Dacker Brands.

- Objectives:
  - Harmonize global data and deliver consolidated reporting.
- Scope:
  - International systems, currency conversions, consolidated KPIs.
- Key Components:
  - Data standardization, ETL pipelines, consolidation logic.
- Technologies:
  - Databricks, Azure, BI tools.
- Deliverables:
  - Consolidated reports, standardized datasets, runbooks.
